# B Tree Implementation

## How to run?
1. `git clone` project link
2. To compile, `clang++ -std=c++11 BTree.cpp -o <name-you-want-to-give>` or `g++ -std=c++11 BTree.cpp -o <name-you-want-to-give>`
3. `./BTree`, i.e. name-you-want-to-give = BTree
